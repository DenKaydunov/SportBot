# 🔥 Система стриков (Streaks)

## Описание

Система стриков отслеживает последовательные дни тренировок пользователя. Это мотивирует пользователей тренироваться ежедневно и формирует полезную привычку.

## Функционал

### Основные возможности:
- ✅ **Автоматическое отслеживание** - стрик обновляется при каждой тренировке
- ✅ **Текущий стрик** - количество дней подряд с тренировками
- ✅ **Лучший стрик** - рекордная серия дней
- ✅ **Уведомления** - сообщения при обновлении стрика
- ✅ **Отображение в профиле** - информация о стрике в профиле пользователя

## Логика работы

### Обновление стрика:
1. **Первая тренировка** - стрик устанавливается в 1
2. **Тренировка сегодня, последняя была вчера** - стрик увеличивается на 1
3. **Тренировка сегодня, последняя была сегодня** - стрик не меняется
4. **Тренировка сегодня, последняя была раньше вчера** - стрик сбрасывается до 1
5. **Тренировка в прошлом** - стрик пересчитывается на основе предыдущих тренировок

### Лучший стрик:
- Автоматически обновляется, когда текущий стрик превышает предыдущий рекорд
- Сохраняется даже если текущий стрик сброшен

## API Endpoints

### Получить информацию о стрике
```
GET /api/v1/streak?telegramId={telegramId}&lang={lang}
```
Возвращает форматированную строку со стриком.

**Пример ответа:**
- `🔥 Стрик: 15 дней подряд (рекорд: 20 дней)`
- `🔥 Стрик: 5 дней подряд (новый рекорд! 🎉)`
- `🔥 Стрик потерян. Лучший результат: 10 дней. Прошло дней без тренировки: 3`

### Получить текущий стрик
```
GET /api/v1/streak/current?telegramId={telegramId}
```
Возвращает число - текущий стрик.

### Получить лучший стрик
```
GET /api/v1/streak/best?telegramId={telegramId}
```
Возвращает число - лучший стрик.

## Интеграция

### В профиле пользователя
Стрик автоматически отображается в профиле пользователя через `UserProfileService`.

### При сохранении тренировки
При сохранении результата тренировки через `ExerciseService` автоматически:
1. Обновляется стрик пользователя
2. Добавляется сообщение о стрике в ответ (если стрик изменился)

**Пример сообщения:**
```
Отжимания: сделано 50 повторений. Общее число: 1500.
🔥 Стрик: 5 дней подряд!
```

## База данных

### Таблица `users`
Добавлены поля:
- `current_streak` (INTEGER, NOT NULL, DEFAULT 0) - текущий стрик
- `best_streak` (INTEGER, NOT NULL, DEFAULT 0) - лучший стрик
- `last_workout_date` (DATE) - дата последней тренировки

### Миграция
Миграция находится в `db/changelog/changes/0008-add-streaks.sql`

## Локализация

Сообщения о стриках находятся в `messages.properties`:
- `streak.no_workouts` - нет тренировок
- `streak.active` - активный стрик
- `streak.active_record` - новый рекорд
- `streak.lost` - стрик потерян
- `workout.streak_updated` - уведомление об обновлении стрика

## Использование в коде

### Обновление стрика
```java
@Autowired
private StreakService streakService;

// При сохранении тренировки
streakService.updateStreak(user, LocalDate.now());
```

### Получение информации о стрике
```java
String streakInfo = streakService.getStreakInfo(user, Locale.forLanguageTag("ru"));
int currentStreak = streakService.getCurrentStreak(user);
int bestStreak = streakService.getBestStreak(user);
```

## Примеры использования

### Telegram Bot
```java
// Команда /streak
String streakInfo = streakService.getStreakInfo(user, locale);
sendMessage(chatId, streakInfo);
```
