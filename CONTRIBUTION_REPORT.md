- **Безопасность (TSP-405)** — защита от утечки секретов в Git
- **Инфраструктура** — настройка CI/CD (GitHub Actions), конфигурация GCP, рефакторинг локализации
- **Документация** — создание CLAUDE.md с полным описанием архитектуры проекта
